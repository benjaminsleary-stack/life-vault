# Skill: charlotte-surfacer

Surface **one** real, timely thing about Charlotte (or the kids) that Ben captured,
so he remembers it. This is the emotional core — get the restraint right.

## Binding rules
- **Captured facts only.** Use only fragments actually written in `people/charlotte.md`
  (and `people/kids.md`). Never infer, generalise, or give generic relationship advice.
- **Cite the date** the thing was captured.
- **Timely first.** Prefer fragments tied to something soon (an event, an operation,
  a stressful week) or a thread left open.
- **Don't repeat.** Skip any fragment stamped `surfaced:` within the last **14 days**.
- **One item, max.** If nothing qualifies, return nothing — silence is fine.

## Steps
1. Read `people/charlotte.md` (then `kids.md` if Charlotte has nothing eligible).
2. Pick the single most useful eligible fragment. Phrase it as a gentle prompt,
   e.g. *"Charlotte's mum's op is on the 14th (noted 10 Jul) — worth checking in."*
3. Stamp that fragment in the log with `surfaced: <today>` so it's excluded for 14 days.
4. Return just the one line (the morning brief places it). Commit the `surfaced:`
   stamp with the brief's push.

Never surface something that would feel like surveillance or a nag. When in doubt,
surface nothing.
